# TransCoder - Inicio Rápido (2 minutos)

¿Primera vez con TransCoder? Esta guía te llevará de 0 a crear tu primer proyecto optimizado en 2 minutos.

---

## Paso 1: Instalación (30 segundos)

```bash
cd TransCoder
python install.py
```

Espera a ver: `✅ ¡Instalación completada!`

---

## Paso 2: Demostración Interactiva (1 minuto)

```bash
python quickstart.py
```

Esto te mostrará:
- 📁 Estructura del proyecto
- 📋 Ejemplo de proyecto real
- 📝 Comparación de prompts (con/sin TransCoder)
- 💰 Ahorro de tokens (~65%)

---

## Paso 3: Tu Primer Proyecto (30 segundos)

```bash
python transcoder-cli.py
```

Responde 4 preguntas simples:
1. **Nombre:** (ej: "mi-dashboard")
2. **Tipo:** 1=Web, 2=App, 3=API
3. **Stack:** (ej: "Next.js, TypeScript")
4. **Feature:** (ej: "Dashboard con gráficas")

TransCoder automáticamente:
- ✅ Genera plan completo (5-7 pasos)
- ✅ Crea prompt optimizado para paso 1
- ✅ Lo copia a tu portapapeles

---

## Paso 4: Usar el Prompt

1. Abre **Claude Code** (o tu agente favorito)
2. **Pega** el prompt (Ctrl+V / Cmd+V)
3. El agente ejecutará perfectamente el paso
4. Vuelve a TransCoder y marca como completado
5. Repite para siguientes pasos

---

## Bonus: Uso con Claude Code (Skill)

Si tienes Claude Code instalado, es aún más simple:

```
Tú: "Nuevo proyecto con transcoder: API de tareas con FastAPI"

Claude Code: [Automáticamente activa TransCoder y te guía]
```

## Comandos Útiles

```bash
# Ver estado del proyecto
python transcoder-cli.py status

# Siguiente paso
python transcoder-cli.py next

# Auditar eficiencia
python transcoder-cli.py audit
```
